R-Package Repository

Description

'drats' are personal R package repositories hosted via Github and using the drat package to and petermeissner/dratful to learn more.

Installation

To install from an personal R package repository like this, use the following code:

    install.packages(
      pkgs  = "package_name",
      repos =
        c(
          options("repos")$repos,
          "path_to_repository"
        )
    )
